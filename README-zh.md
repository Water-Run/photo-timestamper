# 照片时间水印添加器  

*[English](./README.md)*  

![软件图标, 使用NanoBananaPro生成](assets/readme-logo.png)

`照片时间水印添加器`(`photo-timestamper`)是一个简单的桌面端应用, 用于为照片添加**模拟相机原厂风格**的时间水印.

![示例图, 使用`佳能`预设](./assets/demo.png)

*使用PyQT WebEngine, 并开源于[GitHub](https://github.com/Water-Run/photo-timestamper).*

***[下载](https://github.com/Water-Run/photo-timestamper/releases/tag/1.0)*** `photo-timestamper-[你的系统].zip`并解压后, 运行`install.bat`(Windows)或`install.sh`即可安装.  

## 水印风格  

`照片时间水印添加器`预设了契合相机原厂风格的多个预设水印:

- `佳能`  

- `尼康`  

- `索尼`  

- `松下`  

- `富士`  

- `宾得`  

- `小米`

- `经典胶片`  

> 可以通过向`style/`下添加符合规范的`YML`进行增加/修改预设. 使用[SimpSave](https://github.com/Water-Run/SimpSave/tree/master/source)进行格式化存储, 且将数据库命名为`英文&中文`
